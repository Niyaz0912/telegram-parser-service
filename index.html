Telegram Parser

Скрипт для сбора сообщений из Telegram-чатов по ключевым словам.

Скачать архив (ZIP)

Как использовать:

  1. Получите API ключи на my.telegram.org.
  2. Распакуйте архив, настройте config.json.
  3. Запустите python parser.py, введите ссылку на чат и ключевые слова.
  4. Результаты в файле messages.csv.
